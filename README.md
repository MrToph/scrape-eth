# scrape-eth

> Recursively scrapes a website for all ETH addresses and pulls contract name information from etherscan/bscscan.

Supported networks:
- ETH mainnet
- BSC mainnet


## Install

```bash
$ npm install --global scrape-eth
```


## CLI

```
$ scrape-eth --help
$ scrape-eth --url yearn.finance [--chain bsc]
```
